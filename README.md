# soundora

Soundora est une boutique en ligne destinée aux musiciens amateurs et professionnels, leur permettant d’acheter des instruments de musique et accessoires facilement. L’application doit offrir une expérience intuitive, inspirante et adaptée aux besoins des utilisateurs qui souhaitent découvrir, comparer, et acheter du matériel musical en ligne, depuis leur smartphone ou ordinateur, en toute autonomie ou avec assistance. L’application peut être utilisée à la maison, en studio, ou en déplacement, et doit inclure des fonctionnalités interactives (démos, recommandations, vidéos)

Fonctionnalitées de la boutique :

    Affichage des produits ( images, prix, description ) 
    Affichage des stocks
    Filtrage ( par prix, produits).

🧺 Panier

    Ajouter / supprimer un produit avec affichage de ce dernier.
    Calcul et affichage du prix total

👤 utilisateur

    Connexion et création d'un compte sécurisée (JWT)
    Création d'un espace personnel avec informations de l'utilisateur , historique des commandes.
    Pouvoir se connecter et se déconnecter.

🛠️ Back-office

    Gestion des produits (CRUD)
    Suivi des commandes
    Interface réservée aux administrateurs


💳 Paiement sécurisé

    Paiement par carte bancaire.
    Confirmation après paiement par message toast.
    Sécurité des données de transaction

🔐 Sécurité

    Authentification par jeton JWT via Supabase


🧰 Stack technique :

Frontend : 	Angular, HTML /CSS
Backend : Express.js (Node.js)
Base de données : Supabase
Authentification 	JWT (JSON Web Tokens) via Supabase.
Responsive 	en HTML / CSS
