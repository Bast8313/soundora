# Images du site Soundora

## Bannières principales (3 images)

Placer vos images ici avec les noms suivants :

- `banner-guitares.jpg` (600x250px recommandé)
- `banner-basses.jpg` (600x250px recommandé)  
- `banner-effets.jpg` (600x250px recommandé)

**Format recommandé :** JPG ou PNG
**Dimensions recommandées :** 600x250 pixels
**Poids max recommandé :** 200 Ko par image

## Images temporaires

Pour l'instant, le site utilise des images placeholder (via.placeholder.com).
Une fois que vous aurez ajouté vos images ici, modifiez le fichier :
`src/app/components/banner-images/banner-images.component.ts`
